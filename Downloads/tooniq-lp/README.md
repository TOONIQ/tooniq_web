# 合同会社TOONIQ コーポレートサイト

## 概要
合同会社TOONIQの企業ホームページです。AIとアニメ制作を融合させた革新的なサービスを提供する企業の公式サイトです。

## 技術スタック
- **フレームワーク**: Next.js 14 (App Router)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS
- **アニメーション**: Framer Motion
- **アイコン**: Lucide React

## 機能
- ✅ レスポンシブデザイン（モバイル/タブレット/デスクトップ対応）
- ✅ ダークモード対応
- ✅ スムーズなアニメーション
- ✅ SEO最適化
- ✅ 高速パフォーマンス

## 限界を超えた機能
- 🌌 インタラクティブなパーティクル背景
- ✨ 3Dカード効果とホログラフィックデザイン
- 🎯 カスタムカーソル（デスクトップ）
- 🌟 グローイングテキストエフェクト
- 📝 タイプライターアニメーション
- 🎨 パララックススクロール効果
- ⚡ スムーススクロール体験
- 🚀 没入型ローディング画面
- 🖼️ Unsplashの高品質画像統合

## ページ構成
1. **ホーム** - ヒーローセクション、ミッション、サービス概要
2. **会社概要** - 企業情報、チーム紹介、沿革
3. **サービス** - 6つのサービスの詳細説明
   - AnimaTime（クラウド制作進行ソフト）
   - 技術コンサルタント・顧問
   - ChatBot作成
   - 議事録ツール
   - カスタム開発
   - Webサイト作成・運用代行
4. **実績** - プロジェクト実績と成果
5. **お問い合わせ** - コンタクトフォーム
6. **採用情報** - 外部リンク（https://tooniq.co.jp/recruit）

## セットアップ

### 必要な環境
- Node.js 18.x 以上
- npm 9.x 以上

### インストール
```bash
# 依存関係のインストール
npm install
```

### 開発サーバーの起動
```bash
npm run dev
```
ブラウザで http://localhost:3000 にアクセス

### ビルド
```bash
npm run build
```

### 本番サーバーの起動
```bash
npm run start
```

## プロジェクト構造
```
tooniq-lp/
├── app/                    # Next.js App Router
│   ├── layout.tsx         # ルートレイアウト
│   ├── page.tsx           # ホームページ
│   ├── globals.css        # グローバルCSS
│   ├── about/             # 会社概要ページ
│   ├── services/          # サービス一覧ページ
│   ├── works/             # 実績紹介ページ
│   └── contact/           # お問い合わせページ
├── components/            # 共通コンポーネント
│   ├── Header.tsx         # ヘッダー
│   ├── Footer.tsx         # フッター
│   └── ThemeProvider.tsx  # ダークモードプロバイダー
├── public/                # 静的ファイル
└── package.json           # プロジェクト設定
```

## カスタマイズ

### カラーテーマ
`tailwind.config.ts` でカスタムカラーを定義：
- anime-purple: #8B5CF6
- anime-blue: #3B82F6
- anime-pink: #EC4899
- anime-cyan: #06B6D4

### ダークモード
デフォルトでダークモードが有効。ヘッダーのトグルボタンで切り替え可能。
